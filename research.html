Projects during B.Tech

  1. Fingerprint Verification System (B.Tech final year project)
    Guided by Prof J. N. Sarvaiya
    Researched about various algorithms used in various steps of fingerprint verification like gabor filtering etc. Finally, developed a matlab code to implement a complete fingerprint verification algorithm.


  2. Text to Speech Conversion(Summer Internship project in IIT, Delhi)
    Guided by Prof Brijesh Lal
    A TTS library (flite) was build on the processor TMS320C5510. It involved installing dsp/bios kernel on the DSP processor and then porting the library onto it. The project involved detailed study of TMS320C55x processors and the how a text to speech library is built.


  3. Swarm Robotics
    Built a pair of robots which communicated wirelessly. Also, these robots could traverse grid and were used to implement a specific set of instruction that were the part of the game.
